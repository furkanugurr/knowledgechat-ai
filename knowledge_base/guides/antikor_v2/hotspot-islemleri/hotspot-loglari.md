# Hotspot Logları

## Kapsam

Hotspot kullanıcılarının loglarının görüntülendiği bölümdür.

## Kullanım adımları

1. Kullanıcı adı veya IP adresi ile filtrelenebilmektedir.

## Alanlar

- `Kullanıcı Adı` (Hotspot Logları paneli): Hotspot kullanıcı adını girmek için.
- `IP Adresi` (Hotspot Logları paneli): Hotspot IP adresini girmek için.
- `Kullanılan Süre` (section_name): Kullanılan süreyi girin.
- `Kullanılan Boyut` (section_name): Kullanılan boyutu girin.
- `Kullanılan IP Adresleri` (section_name): Kullanılan IP adreslerini girin.
- `Giriş Zamanı` (section_name): Giriş zamanını girin.
- `Çıkış Zamanı` (section_name): Çıkış zamanını girin.

## Görünür kontroller

- `XLS`: Excel dosyası oluşturmak için.
- `CSV`: CSV dosyası oluşturmak için.
- `PDF`: PDF dosyası oluşturmak için.
- `Göster/Gizle`: Gösterecek veya gizlenecek verileri seçmek için.
- `Sayfa Başı Kayıt Sayısı`: Sayfada gösterilecek kayıtların sayısını ayarlamak için.
- `Tamam`: Gösterilen verileri onaylamak için.
- `Filtrele`: Verileri filtrelemek için.
- `Filtreyi Temizle`: Filtreleri temizlemek için.
