# Yönetim Paneli Ayarları

## Kapsam

Antikor2 NGFW Web yönetim arabirimine ait Servis Ayarları ve Servis Durumları konfigürasyonu buradan yapılmaktadır.

## Kullanım adımları

1. Web Arayüzü Erişim Protokolü seçimi yapın.
2. Servis Portu Numarası girin.
3. Yönetim Paneli için Cpu Rezervasyonu ve Bağımsız Yönetim Altyapısı seçeneklerini ayarlayın.
4. Atanacak Ethernet seçeneğini belirleyin.
5. IP Adresi ve Varsayılan Ağ Geçidi değerlerini girin.
6. Güncelleme için Çıkış Portu seçeneğini belirleyin.
7. Kaydet butonuna tıklayın.
8. Ethernet Atama haricinde Ethernetlerin Web arayüz erişimlerini yönetmek için bu sayfayı kullanın.

## Alanlar

- `Web Arayüzü Erişim Protokolü` (Servis Ayarları paneli): HTTPS veya HTTP protokolünü seçer.
- `Servis Portu Numarası` (Servis Ayarları paneli): Web arayüzünün erişim portunu belirler.
- `Atanacak Ethernet` (Servis Ayarları paneli): Ethernet atamasını seçer.
- `IP Adresi` (Servis Ayarları paneli): IPv4 adresini belirler.
- `Varsayılan Ağ Geçidi` (Servis Ayarları paneli): IPv4 ağ geçidini belirler.
- `Güncelleme için Çıkış Portu` (Servis Ayarları paneli): Güncelleme için çıkış portunu seçer.
- `Ethernet Adı` (Ethernetlerin Web arayüz erişimleri): Ethernet portunun adını gösterir.
- `Arayüz` (Ethernetlerin Web arayüz erişimleri): Ethernet portunun hangi arayüze ait olduğunu gösterir.
- `Web Arayüz Erişimi` (Ethernetlerin Web arayüz erişimleri): Ethernet portunun web arayüzüne erişimini açıp kapattığını gösterir.

## Görünür kontroller

- `Servis Ayarları`: Web yönetim arayüzünün erişim protokolü ve port numarasını belirler.
- `Yönetim Paneli için Cpu Rezervasyonu`: CPU rezervasyonunun açık/kapalı durumunu ayarlar.
- `Bağımsız Yönetim Altyapısı`: Bağımsız yönetim altyapısının açık/kapalı durumunu ayarlar.
- `Kaydet`: Değişiklikleri kaydeden ve ekrana geri dönmenin yolunu açan buton.
- `Yenile`: Servis durumlarını yeniden yükleme için kullanılır.

## Uyarılar

- Web yönetim arayüzü ile güvenli iletişimi sağlamak için TLS1.2/SSL kullanılmaktadır.
