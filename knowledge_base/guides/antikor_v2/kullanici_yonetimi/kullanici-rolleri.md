# Kullanıcı Rolleri

## Kapsam

Kullanıcılara ait roller Ekle butonu ile tanımlanabilmektedir.

## Menü yolu

- `Kullanıcı Yönetimi > Kullanıcı Rolleri`
- `Yetkiler > Tanımlamalar`
- `Rollere ait tanımlar tek başına seçilebildiği gibi seçilen nesnenin (görüntle, ekle, güncelle, sil) ayrıca detaylı olarak tanımlanabilmektedir. Admin haricindeki diğer Antikor yöneticilerinin tanımları uygula yapabilmesi için “Uygulanacak İşlem Listesi” seçeneğinin de seçili olması gereklidir.`

## Kullanım adımları

1. Adı ve açıklama alanlarını doldurun.
2. Kaydet butonuna tıklayın.
3. Hepsi > Tanımlamalar > Ağ Tanımları > Görüntüle
4. Hepsi > Tanımlamalar > Hotspot İstemcileri > Ekle
5. Hepsi > Tanımlamalar > İstemci Grupları > Güncelle
6. Hepsi > Tanımlamalar > İstemci Tanımları > Sil
7. Uygulanacak İşlem Listesi seçeneğini seçin.

## Alanlar

- `Sayfa Başı Kayıt Sayısı` (header section): Sayfa Başı Kayıt Sayısı
- `Adı` (panel): Kullanıcı rolünün adını girin.
- `Açıklama` (panel): Kullanıcı rolünün açıklamasını girin.

## Görünür kontroller

- `+ Ekle`: + Ekle
- `Göster/Gizle`: Show/hide columns dropdown menu
- `Tamam`: Confirm button for actions
- `Filtrele`: Filter button to filter user roles
- `Filtreyi Temizle`: Clear filters button for user roles
- `İptal`: Ekrandaki işlemleri iptal etmek için kullanılır.
- `Kaydet`: Eklenecek kullanıcı rolünün bilgilerini kaydetmek için kullanılır.
- `Hepsi`: Yetkilendirme kapsamlarını seçer
- `Tanımlamalar`: Yetki tanımları için alt menülerin seçimi
- `Ağ Tanımları`: Ağ yetkilerini seçer
- `Görüntüle`: Yetki listesini görüntüler
- `Ekle`: Yeni yetki ekler
- `Güncelle`: Yetkinin bilgilerini günceller
- `Sil`: Yetkinin silinmesi için kullanılır
- `Hotspot İstemcileri`: Hotspot yetkilerini seçer
- `İstemci Grupları`: İstemci grup yetkilerini seçer
- `İstemci Tanımları`: İstemci tanımlarını seçer
- `Kimlik Tanımlamaları`: Kimlik tanımlarını seçer
- `Özel Kullanıcılar`: Özel kullanıcı yetkilerini seçer
- `Port Tanımları`: Port tanımlarını seçer
- `Uç Tanımları`: Uç tanımlarını seçer
- `Sistem Ayarları`: Sistem ayarlarını seçer
- `Ağ Yapılandırması`: Ağ yapılandırmasını seçer
- `Uygulanacak İşlem Listesi`: Admin haricindeki diğer Antikor yöneticilerinin tanımları uygula yapabilmesi için seçilmelidir.
