# Kimlik Doğrulama Test Ekranı

## Kapsam

İstemciler için servisin adı seçilerek kimlik doğrulama test edilebilmektedir.

## Menü yolu

- `Kimlik Doğrulama Test Ekranı`

## Kullanım adımları

1. Servis Adı seçeneğini kullanarak istemciler için servisin adını seçin.
2. Servis Adı seçeneğini kullanarak istemci için servis adını seçin.
3. Seçilen servis adı ile kimlik doğrulama test edin.

## Alanlar

- `Servis Adı` (Kimlik Doğrulama Test Ekranı): Seçim yapmadan önce 'Seçiniz...' yazısı görünür. Seçim yapıldığında servisin adı listesinden biri seçilebilir.

## Görünür kontroller

- `Doğrula`: Servisin adını seçtiğinizde doğrulama testi yapmak için kullanılır.
- `Servis Adı`: İstemciler için servisin adını seçmek için kullanılan menü.
