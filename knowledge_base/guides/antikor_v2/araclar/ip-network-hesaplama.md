# IP - Network Hesaplama

## Kapsam

Bu bölümde IP Network Hesaplama yapılabilmektedir. Aşağıda bir örnek görülmektedir. Sağ tarafta IP adresleri kapsam içerisinde olup olmadığı kontrolleri yapılabilir.

## Kullanım adımları

1. Network Hesaplama panelinde 'Hesapla' butonuna tıklayın.

## Alanlar

- `Network` (Network Hesaplama paneli): IPv4 adresi ve maske bit değeri için kullanılabilir IP aralığı hesaplanır.
- `Maske Bitleri` (Network Hesaplama paneli): IPv4 adresi ve maske bit değeri için kullanılabilir IP aralığı hesaplanır.
- `Alt Ağ Maskesi` (Network Hesaplama paneli): IPv4 adresi ve maske bit değeri için kullanılabilir IP aralığı hesaplanır.
- `Başlangıç IP` (Network Hesaplama paneli): IPv4 adresi ve maske bit değeri için kullanılabilir IP aralığı hesaplanır.
- `Bitiş IP` (Network Hesaplama paneli): IPv4 adresi ve maske bit değeri için kullanılabilir IP aralığı hesaplanır.
- `Network Adres` (Network Hesaplama paneli): IPv4 adresi ve maske bit değeri için kullanılabilir IP aralığı hesaplanır.
- `Broadcast Adres` (Network Hesaplama paneli): IPv4 adresi ve maske bit değeri için kullanılabilir IP aralığı hesaplanır.

## Görünür kontroller

- `Hesapla`: Network Hesaplama için hesaplamaları başlatır.
- `Kontrol Et`: IP Adresi'nin kapsam kontrolünü yapar.
