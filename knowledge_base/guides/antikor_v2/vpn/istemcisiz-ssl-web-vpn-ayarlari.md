# İstemcisiz SSL Web VPN Ayarları

## Kapsam

Bu kılavuz, İstemcisiz SSL Web VPN Ayarları hakkında bilgi sunmaktadır.

## Menü yolu

- `İstemcisiz SSL Web VPN Ayarları > Profiller ve Kaynaklar > Kullanıcı Profil Atama`
- `Profiller ve Kaynaklar > Yeni Kayıt`

## Kullanım adımları

1. Default profili düzenleme için Düzenle butonuna tıklayın.
2. Default profili silmek için Sil butonuna tıklayın.
3. Ad alanına yeni profil için ad girin.
4. Kaynaklar seçeneğinden profil oluşturmak istediğiniz kaynaktan birini seçin.

## Alanlar

- `Kullanıcı Adı` (Kullanıcı Profil Atama paneli): Kullanıcı Profil Atama panelinde kullanıcı adı girilecek alan
- `Profiller` (Kullanıcı Profil Atama paneli): Kullanıcı Profil Atama panelinde profiller seçilecek alan
- `Ad` (Ad): Yeni profil için ad girin.
- `Kaynaklar` (Kaynaklar): Profil oluşturmak istediğiniz kaynaktan birini seçin.
- `Kullanıcı Adı` (User Profile Section): Dropdown for selecting a username.
- `Profiller` (User Profile Section): Dropdown for selecting user profiles.

## Görünür kontroller

- `Kaynaklar`: Kaynaklar paneli
- `Yenile`: Profiller ve Kaynaklar panelinde yeniden yükleme
- `+ Ekle`: Profiller ve Kaynaklar panelinde yeni kaynak ekleme
- `Düzenle`: Default profili düzenleme
- `Sil`: Default profili silme
- `İptal`: Eğer işlemi iptal etmek isterseniz bu düğme tıklayın.
- `Kaydet`: Girilen bilgileri kaydedin ve profil oluşturun.

## Uyarılar

- Model aynı etiketi control ve field olarak sınıflandırdı; yapılandırma değeri olan field korundu: Profiller
