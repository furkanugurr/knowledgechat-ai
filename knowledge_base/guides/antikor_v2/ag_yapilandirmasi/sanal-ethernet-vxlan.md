# Sanal Ethernet - VXLAN

## Kapsam

VXLAN (Virtual Extensible LAN), sanal ağları daha geniş veri merkezlerinde veya bulut ortamlarında ölçeklendirebilmek için kullanılan bir teknolojidir. Bu teknoloji, özellikle büyük veri merkezleri ve bulut ortamlarında sanal makinelerin (VM) farklı fiziksel sunucularda olmasına rağmen sanki aynı yerel ağda (LAN) bulunuyormuş gibi birbirleriyle iletişim kurmalarını sağlar.

## Kullanım adımları

1. Adı alanına sanal Ethernet - VXLAN'ın adını girin.
2. VNI - VXLAN Ağ Kimliği alanına VXLAN ağ kimliğini girin.
3. Kaynak IP alanına kaynak IP adresini girin.
4. Hedef IP alanına hedef IP adresini girin.
5. Kaynak IP adresini girin.
6. Hedef IP adresini girin.
7. Açıklama alanına gerekli bilgileri girin.
8. Kaydet butonuna tıklayın.

## Alanlar

- `Adı` (Sanal Ethernet - VXLAN paneli): Sanal Ethernet - VXLAN'ın adını girin
- `VNI - VXLAN Ağ Kimliği` (Sanal Ethernet - VXLAN paneli): VXLAN ağ kimliğini girin
- `Kaynak IP` (Sanal Ethernet - VXLAN paneli): Kaynak IP adresini girin
- `Hedef IP` (Sanal Ethernet - VXLAN paneli): Hedef IP adresini girin
- `Açıklama` (Sanal Ethernet - VXLAN): Ağ için açıklama yapın. Bu, VXLAN ağının ne işe yaradığını açıklayabilir.

## Görünür kontroller

- `Yenile`: Listeyi yenileyin
- `Ekle`: Yeni bir kaydı ekleyin
- `İptal`: Kaydettiğiniz bilgileri iptal etme ve önceki sayfaya geri dönme.
- `Kaydet`: Girilen VXLAN ağ bilgilerini kaydetme.
