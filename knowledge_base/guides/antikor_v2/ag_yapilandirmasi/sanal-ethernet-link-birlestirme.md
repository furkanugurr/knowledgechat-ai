# Sanal Ethernet - Link Birleştirme

## Kapsam

Antikor’un üstünden geçen hızı artırmak veya Cluster olarak Antikor çalıştırmak için kullanılmaktadır. Birleştirme türü olarak 7 seçenek mevcuttur.

## Kullanım adımları

1. Birleştirme türünü seçin.
2. Bağlantının adını girin.

## Alanlar

- `Birleştirme Türü` (Sanal Ethernet - Link Birleştirme paneli): Bağlantı türünü seçmek için
- `Adı` (Sanal Ethernet - Link Birleştirme paneli): Bağlantının adını girin
- `Span Port` (Sanal Ethernet - Link Birleştirme paneli): Span portunu girin
- `MTU` (Sanal Ethernet - Link Birleştirme paneli): Maximum Transfer Unit değerini girin

## Görünür kontroller

- `Yenile`: Listeyi yeniden yükleme
- `Ekle`: Yeni bir bağlantı eklemek için
- `Göster/Gizle -`: Liste gösterimi veya gizlendirmesi
- `Filtrele`: Listeyi filtrelemek için
- `Filtreyi Temizle`: Filtreleri temizlemek için
