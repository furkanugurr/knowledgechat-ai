# IP Havuzları

## Kapsam

Yönetilecek olan ağ(lar)ın IP adres aralıkları bu menüde tanımlanır. Ethernet atama işlemi gerçekleştirilmeden önce kullanılacak olan IP bloğu buraya eklenmelidir. Bu aralıklar, AntiKor’un denetleme mekanizmalarında kullanılacaktır.

## Menü yolu

- `IP Havuzları`

## Kullanım adımları

1. Ethernet ataması yapın.
2. IP Bloğunu girin.
3. Ethernet portunu seçin.
4. IPv4 veya IPv6 adres ailesini seçin.
5. IPv4 IP bloğunu girin.
6. Açıklamayı girin (isteğe bağlı).

## Alanlar

- `Ethernet` (Ethernet column): Ethernet ataması.
- `IP Bloğu` (IP Bloğu column): IP bloğunu girin.
- `Açıklama` (Açıklama column): Açıklamayı girin.
- `Adres Ailesi` (Adres Ailesi): IPv4 veya IPv6 adres ailesini seçmek için düğme.

## Görünür kontroller

- `+ Ekle`: Yeni IP havuzu ekleme.
- `Göster/Gizle`: Liste gösterimini ayarlamak için.
- `Tamam`: Seçilen filtreleri uygula.
- `Filtrele`: Liste filtrelenmesi için.
- `Filtreyi Temizle`: Filtreleri temizleme.
- `İptal`: Eylemi iptal etmek için.
- `Kaydet`: Kaydedilen bilgileri kaydetmek için.

## Uyarılar

- Model aynı etiketi control ve field olarak sınıflandırdı; yapılandırma değeri olan field korundu: Ethernet
