## Alanlar

- `Adı` (Modem Arayüzü): Name of the modem or interface.
- `Açıklama` (Modem Arayüzü): Description of the modem or interface.
- `Kullanıcı Adı` (Erişim Noktası Adı (APN)): Username for PPP connection.
- `Çevirilecek Numara` (Erişim Noktası Adı (APN)): Number to be converted during the PPP connection process.
- `Erişim Noktası Adı (APN)` (Erişim Noktası Adı (APN)): Access Point Name for PPP connection.
- `İşlemler` (Bağlantı Durumu): Actions related to the modem or interface.
- `Bağlantı Durumu` (Bağlantı Durumu): Connection status of the modem or interface.
- `Modem Tipi` (Modem Arayüzünde): Bağlantı tipini seçer.
- `Parola` (Modem Arayüzünde): Internet servis sağlayıcısının parolayı girer.
- `Ethernet Adı` (Ethernet Atama menüsü): Sanal Ethernet - PPP adını girer. (Bu örnek için ppp1)
- `Web Arayüz Erişimi Aktif` (WAN menüsü): Web arayüzüne erişim aktif edilir.
- `Durum` (Ethernet Durumları): Ethernet atamasının aktif olup olmadığını belirler. (Açıklama: Aktif)
- `Güvenlik Bölgesi` (Ethernet Durumları): Ethernet atamasına ait güvenlik bölgesini seçin.
- `Arayüz` (Ethernet Durumları): Ethernet ataması için kullanılacak arayüzü seçin. (Açıklama: WAN2)
- `Hız` (Ethernet Durumları): Ethernet atamasının hızını seçin. (Açıklama: autoselect)
- `MTU` (Ethernet Durumları): Ethernet atamasına ait MTU değerini girin.
- `Web Arayüzü Erişimi` (Ethernet Durumları): Ethernet atamasının web arayüzüne erişimini aktif eder. (Açıklama: Aktif)
- `Cluster Üyeliği` (Ethernet Durumları): Ethernet atamasının cluster üyelik durumunu belirler.
- `Cluster Ethernet Adı` (Ethernet Durumları): Ethernet atamasına ait cluster ethernet adını girin. (Açıklama:)
- `Modem Arayüzü` (Sanal Ethernet - PPP > Yeni Kayıt): Bağlanan modem/modemlerin listesi.
- `Internet Servis Sağlayıcı` (Sanal Ethernet - PPP > Yeni Kayıt): Sanal Ethernet - PPP için internet servisi sağlayıcısı seçimi.
- `Ülke` (Sanal Ethernet - PPP > Yeni Kayıt): Sanal Ethernet - PPP'ye verilecek ülke bilgisi.
- `Sağlayıcı` (Sanal Ethernet - PPP > Yeni Kayıt): Sanal Ethernet - PPP için internet servisi sağlayıcısı seçimi.
- `Plan` (Sanal Ethernet - PPP > Yeni Kayıt): Sanal Ethernet - PPP'ye verilecek plan bilgisi.
- `SIM Kartı PIN` (Sanal Ethernet - PPP > Yeni Kayıt): Sanal Ethernet - PPP için SIM kartı PIN bilgisi.
- `Şebekeyi Algılamak İçin Bekleme Süresi` (Sanal Ethernet - PPP > Yeni Kayıt): Sanal Ethernet - PPP için şebekeyi algılamak için beklenmesi gereken sürenin belirlenmesi.
- `NAT` (Sanal Ethernet - PPP > Yeni Kayıt): Sanal Ethernet - PPP için NAT (Network Address Translation) özelliğinin aktif olup olmadığını belirler.
- `İhtiyaç Halinde Kullan` (Sanal Ethernet - PPP > Yeni Kayıt): Sanal Ethernet - PPP için ihtiyac halinde kullanılması gerektiğini belirtir.
- `Boşta Kalma Zaman Aşımı` (Sanal Ethernet - PPP > Yeni Kayıt): Sanal Ethernet - PPP için boşta kalma zaman aşımının belirlenmesi.
- `İnternet Servis Sağlayıcı` (internet Servis Sağlayıcı): Bağlantı için kullanılacak internet servisi sağlayıcısını seçmek için kullanılan seçenek
- `Şebekeyi Algılamak için Bekleme Süresi` (Şebekeyi Algılamak için Bekleme Süresi): Sunucu yapılandırma opsiyonlarına göre kullanıcı bir bekleme süresi belirleyebilir veya bu alan boş bırakılabilir. Boş bırakılması halinde bekleme süresi “0” olarak belirlenir.

## Görünür kontroller

- `Yenile`: Refreshes the list.
- `Ekle`: Ekle
- `Bağlantı Ekle`: Modem/modemlerin bağlanmasını başlatır.
- `Tanımları Uygula`: Uygulamaların tanımlarını uygular.
- `Hepsini Uygula`: Tüm tanımların uygulanmasını sağlar.
- `Uygula`: Seçilen Sanal Ethernet PPP ayarlarını uygular.
- `İptal`: Eğer işlemi iptal etmek isterseniz bu butona tıklayın.
- `Kaydet`: Ethernet Ataması yapıldıktan sonra bu butona tıklayarak kaydedin.
- `Durum`: Sanal Ethernet - PPP durumu aktif olup olmadığını belirler.
- `Modem Arayüzü`: Modem arayüzünü seçer.
- `İnternet Servis Sağlayıcı`: Bağlantı sağlayıcısını seçmek için kullanılan dropdown menü.
- `Plan`: Kullanılan internet planını seçmek için kullanılan dropdown menü.
- `+ Ekle`: Adds a new virtual Ethernet connection.
- `Düzenle`: Edits an existing virtual Ethernet connection.
- `Sil`: Deletes an existing virtual Ethernet connection.

## Uyarılar

- Tanımları Uygula, unutulmamalıdır.
- Ethernet Atama ayarları yapıldıktan sonra mutlaka tanımlar uygulanmalıdır.
- Oluşturulan Sanal Ethernet - PPP için Açıklama girilir. (Zorunlu Alan)
- 3G/4G modeminizin var ise SIM kartı PIN girilir, yoksa boş bırakılabilir.
- Şebekeyi algılamak için bekleme süresi , sunucu yapılandırma opsiyonlarına göre kullanıcı bir bekleme süresi belirleyebilir veya bu alan boş bırakılabilir. Boş bırakılması halinde bekleme süresi “0” olarak belirlenir.
- İhtiyaç Halinde Kullan varsayılanda pasif olarak gelmektedir. Aktif edildiği takdirde kullanıcıdan Boşta Kalma Zaman Aşımı girilmesi gerekmektedir. Girilmediği takdirde “0” olarak belirlenir. Bu alan kullanıcıların modem kullanılmadığı takdirde gereksiz internet kullanımını ortadan kaldırmak için tasarlanmıştır. Boşta kalan zaman aşımını geçtiği takdirde modem ile bağlantı boşta kaldığı süre boyunca kullanılmayacaktır.
- Tanımları Uygula unutulmamalıdır.
