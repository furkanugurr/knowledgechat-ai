# Ethernet Atama

## Menü yolu

- `Ethernet Atama > DMZ > Yeni Kayıt`
- `Ethernet Atama > PPPoE Ekle Yeni Kayıt`
- `Ethernet Durumları > IP Ayarları`
- `IP Ayarları > Kullanıcı Bilgileri`

## Kullanım adımları

1. Ethernet Durumları panelindeki 'Aktar' butonlarını kullanarak durumu, güvenlik bölgesini ve arayüzü aktarın.
2. Güvenlik Bölgesi, Arayüz, Ethernet Adı, MTU, VRF Adı alanlarını doldurun.
3. Web Arayüzü Erişimi ve Cluster Üyeliği alanlarını 'Aktar' butonları ile aktarın.
4. Cluster Ethernet Adı ve Açıklama alanlarını doldurun.
5. IPv4 Ayarları panelinde IPv4 adresi, DHCPv4 havuzu modu, başlangıç, bitiş, ağ geçidi ve relay adresini girin.
6. Kira süresini belirleyin.
7. IPv6 Ayarları panelinde IPv6 adresi, DHCPv6 başlangıç, bitiş ve relay adresini girin.
8. Ethernet portunun durumunu seçin.
9. Güvenlik bölgesini seçin.
10. Arayüzü seçin.
11. Ethernet adını belirleyin.
12. MTU değerini belirleyin.
13. VRF adını belirleyin.
14. Web arayüz erişimini aktif edin veya devre dışı bırakın.
15. Cluster üyeliğini aktif edin veya devre dışı bırakın.
16. Otomatik IPv4 AI seçeneğini kullanmak için 'Otomatik AI' tabını seçin.

## Alanlar

- `Açıklama` (Ethernet Durumları): Ethernet durumunu açıklar.
- `IPv4 Adresi` (IP Ayarları): Ethernet arayüzünün IPv4 adresini belirler.
- `IPv6 Adresi` (IP Ayarları): Ethernet arayüzünün IPv6 adresini belirler.
- `DHCPv4 Havuzu Modu` (IPv4 Ayarları): DHCPv4 havuzu modunu seçer.
- `DHCPv4 Başlangıç` (IPv4 Ayarları): DHCPv4 başlangıç adresini girer.
- `DHCPv4 Bitiş` (IPv4 Ayarları): DHCPv4 bitiş adresini girer.
- `DHCPv4 Ağ Geçidi` (IPv4 Ayarları): DHCPv4 ağ geçidini girer.
- `DHCPv4 Relay Adresi` (IPv4 Ayarları): DHCPv4 relay adresini girer.
- `Kira Süresi` (IPv4 Ayarları): Ethernet atamasının kira süresini belirler.
- `DHCPv6 Başlangıç` (IPv6 Ayarları): DHCPv6 başlangıç adresini girer.
- `DHCPv6 Bitiş` (IPv6 Ayarları): DHCPv6 bitiş adresini girer.
- `DHCPv6 Relay Adresi` (IPv6 Ayarları): DHCPv6 relay adresini girer.
- `Cluster Ethernet Adı` (Cluster Ethernet Adı): Ethernet arayüzünün cluster adını belirler.
- `Parola` (Kullanıcı Bilgileri paneli): Password text field for user input.
- `Kullanıcı Adı` (Kullanıcı Bilgileri paneli): Username text field.

## Görünür kontroller

- `Güvenlik Bölgesi`: Ethernet güvenlik bölgesini seçer.
- `Arayüz`: Ethernet arayüzü seçer.
- `Ethernet Adı`: Ethernet adını belirler.
- `MTU`: Maximum Transfer Unit değerini belirler.
- `VRF Adı`: Virtual Routing and Forwarding adını belirler.
- `Web Arayüzü Erişimi`: Ethernet arayüzünün web arayüzü erişimini aktif eder veya devre dışı bırakır.
- `Cluster Üyeliği`: Ethernet arayüzünün cluster üyeliğini aktif eder veya devre dışı bırakır.
- `Web Arayüzü Erişimi`: Ethernet arayüzünün web arayüzü erişimini aktif eder veya devre dışı bırakır.
- `Durum`: Ethernet portunun aktif olup olmadığını belirler.
- `Otomatik IPv4 AI`: IPv4 adresi otomatik olarak atanmasını sağlar.
- `Statik`: IPv4 adresini statik olarak ayarlamayı sağlar.

## Uyarılar

- Ethernet portunun durumunu doğru olarak ayarlayın.
- Model aynı etiketi control ve field olarak sınıflandırdı; yapılandırma değeri olan field korundu: Kullanıcı Adı, Parola
