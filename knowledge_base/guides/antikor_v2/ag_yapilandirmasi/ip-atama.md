# IP Atama

## Kapsam

Sistemde farklı veya aynı ethernet arayüzlerine birden fazla IP adres(ler)i verilmesi gerekirse bu bölümden istenilen IP adres(ler)i verilebilir.

## Menü yolu

- `IP Atama`

## Kullanım adımları

1. Network seçeneğini kullanarak ağ seçin.
2. IP Adresi alanına IP adresini girin.
3. Durum seçeneğini aktif olarak belirtin.
4. Network seçeneğini WAN1 olarak ayarlayın.
5. Adres Ailesi seçeneğini IPv4 olarak belirleyin.
6. IP Adresi alanına IPv4 IP adresini girin.
7. Cluster Üyeliği seçeneğini aktif olarak belirtin.
8. Açıklama alanını gerekli bilgilerle doldurun veya boş bırakın.

## Alanlar

- `Network` (Network column): Ağ seçimi.
- `IP Adresi` (IP Adresi column): IP adresi girişi.
- `Açıklama` (Açıklama column): Açıklama metni.

## Görünür kontroller

- `Yenile`: Listeyi yeniden yükleme.
- `Ekle`: Yeni IP atama ekleme.
- `Düzenle`: Seçilen IP adresini düzenleyin.
- `SIL`: Seçilen IP adresini silin.
- `Durum`: IP atamasının aktif olup olmadığını belirtir.
- `Network`: Ağ network seçeneğini gösterir.
- `Adres Ailesi`: IPv4 veya IPv6 adres ailesini belirtir.
- `IP Adresi`: IPv4 IP adresini gösterir.
- `Cluster Üyeliği`: Cluster üyelik durumunu belirtir.
