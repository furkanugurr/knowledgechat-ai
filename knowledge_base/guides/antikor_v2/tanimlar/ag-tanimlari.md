# Ağ Tanımları

## Kapsam

Ağ tanımlarının bulunduğu bölümdür. Bu bölüm içinde Ağ Grupları oluşturup, bu gruplara IP adresi, network, IP adresi aralığı veya FQDN tanımlanabilmektedir. Bu tanımlar güvenlik kurallarında, erişim izinlerinde ve özel kullanıcılar bölümünde de kullanılabilmektedir. Ağ grubu eklemek için Ekle butonuna basıldıktan sonra gelecek olan Ağ Tanımları - Yeni Kayıt penceresinden oluşturmak içerik türüne göre istenilen IP Adres(ler)i/Network(ler) tanımlanabilir.

## Menü yolu

- `Ağ Tanımları > Ağ Grupları`
- `Ağ Grupları > Ağ Grupları Yeni Kayıt`
- `Ağ Grupları > Kullanılan Yerler`
- `Ağ Grupları Yeni Kayıt`
- `Ağ Grupları Kayıt Düzeltme`

## Kullanım adımları

1. Ağ gruplarını yönetmek için 'Gruplar' düğmesine tıklayın.
2. Ağ gruplarını arama yapmak için 'Ara' metin kutusuna değer girip enter tuşuna basın.
3. Gruplar menüsünü seçin.
4. Ara metin kutusuna arama yapın.
5. Ağ Grupları alanına grup ismi girin.
6. Açıklama alanına grup hakkında açıklama yapın.
7. Kaydet butonuna tıklayarak bilgileri kaydedin.
8. IP adreslerini satır satır ekleyin.
9. IPv4 adresi ve subnet maskı girin.
10. Açıklama alanına gerekli bilgileri girin.
11. İçerik türü değiştirilememektedir.
12. Sayfa tabını seçin.
13. Alt Bölüm tabını seçin.
14. Kullanıldığı Yer tabını seçin.
15. Detay Türü tabını seçin.
16. Detay İçeriği tabını seçin.
17. Diğer Detay Türü tabını seçin.
18. Diğer Detay İçeriği tabını seçin.
19. Kayıt ID tabını seçin.
20. Git düğmesini tıklayın.
21. Kaydet butonuna tıklayın.
22. Ağ Grupları Kayıt Düzeltme sayfasına gidin.
23. Ağ Gruplarının adını düzenleyin.
24. IPv4 ağ adresini girin ve 'Kaydet' butonuna tıklayın.
25. Başlangıç IP Adresi alanına IPv4 başlangıç IP adresini girin.
26. Bitiş IP Adresi alanına IPv4 bitiş IP adresini girin.
27. Açıklama alanına açıklamayı girin.
28. Kaydet butonuna tıklayarak formu kaydedin.
29. İçerik Türü seçimi yapın.
30. FQDN alanına değer girin.
31. Açıklama alanına gerekli bilgiyi girin.
32. Başlangıç IP Adresi alanına IPv6 başlangıç IP adresini girin.
33. Bitiş IP Adresi alanına IPv6 bitiş IP adresini girin.
34. Kaydet butonuna tıklayarak ekleme işlemini kaydedin.

## Alanlar

- `Ara` (Ağ Grupları paneli): Ağ gruplarını arama yapmak için kullanılan metin kutusu.
- `#` (Ağ Grup Üyeleri paneli): Kayıt numarası.
- `Grup Adı` (Ağ Grup Üyeleri paneli): Ağ grup adı.
- `Açıklama` (Ağ Grup Üyeleri paneli): Ağ grup açıklaması.
- `İçerik Türü` (Ağ Grup Üyeleri paneli): Ağ içerik türü (IPv4, IPv6).
- `İçerik` (Ağ Grup Üyeleri paneli): Ağ içerik adresi.
- `Notlar` (Ağ Grup Üyeleri paneli): Ağ grup notları.
- `Ağ Grupları` (Gruplar - Kayıt Ekleme Formu paneli): Eklenecek ağ gruplarının ismi girilecektir.
- `IP adresleri` (İp adreslerini ekleyiniz): Satır satır IP adreslerini girilen alan.
- `IPv4` (IPv4 - Kayıt Ekleme Formu): Eklenecek IP adresi ve subnet maskı.
- `FQDN` (FQDN paneli): epati.com.tr adı belirtilmiştir.
- `Adı` (Ağ Tanımları paneli): Grup için ad belirler.
- `IPv4 Adresi` (IPv4 - Kayıt Ekleme Formu panelinde 'IPv4' altındaki alan.): IPv4 ağ adresini girin.
- `Başlangıç IP Adresi` (IPv4 Aralığı - Kayıt Ekleme Formu): IPv4 başlangıç IP adresini girin.
- `Bitiş IP Adresi` (IPv4 Aralığı - Kayıt Ekleme Formu): IPv4 bitiş IP adresini girin.

## Görünür kontroller

- `Gruplar`: Ağ gruplarını yönetmek için kullanılan düğme.
- `Kullanıldığı Yerler`: Ağ gruplarının kullanıldığı yerleri gösteren düğme.
- `Ekle`: Yeni ağ grupları eklemek için kullanılan düğme.
- `Düzenle`: Ağ gruplarını düzenlemek için kullanılan düğme.
- `Sil`: Ağ gruplarını silmek için kullanılan düğme.
- `İptal`: Eklenecek grup bilgilerinin iptal edilmesi için kullanılır.
- `Kaydet`: Grup bilgilerinin kaydedilmesi için kullanılır.
- `Sayfa`: Ağ grupları sayfasına geçiş.
- `Alt Bölüm`: Ağ gruplarının alt bölümlerine göre görüntülenmesi.
- `Kullanıldığı Yer`: Ağ grupları kullanıldığı yerlerine göre görüntülenmesi.
- `Detay Türü`: Ağ gruplarının detay türlerine göre görüntülenmesi.
- `Detay İçeriği`: Ağ gruplarının detay içeriklerine göre görüntülenmesi.
- `Diğer Detay Türü`: Ağ gruplarının diğer detay türlerine göre görüntülenmesi.
- `Diğer Detay İçeriği`: Ağ gruplarının diğer detay içeriklerine göre görüntülenmesi.
- `Kayıt ID`: Ağ grupları kayıtlarına göre görüntülenmesi.
- `Git`: Kullanılan yerler listesine bağlantı yapma.
