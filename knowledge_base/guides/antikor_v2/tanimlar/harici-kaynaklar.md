# Harici Kaynaklar

## Kapsam

{width=900 height=233}

## Menü yolu

- `Tanimlar > Harici Kaynaklar`

## Kullanım adımları

1. Aktif durumda olan kaynakları görüntüleyin.
2. Pasif durumda olan kaynakları görüntüleyin.
3. Path alanına JSON Path'ı girin.
4. > düğmesine tıklayın.
5. Henüz bir istek gönderilmedi.

## Alanlar

- `Sayfa Başı Kayıt Sayısı` (Harici Kaynaklar paneli): Sayfa Başı Kayıt Sayısı
- `URL` (Harici Kaynaklar paneli): URL
- `Kimlik Doğrulama` (Harici Kaynaklar paneli): Kimlik Doğrulama
- `JSON Path` (Harici Kaynaklar paneli): JSON Path
- `Adı` (Request paneli): USOM adını içeren metin kutusu.
- `Zaman Aşımı Süresi(GÜN)` (Request paneli): 30 gününü içeren sayısal kutu.
- `Metot` (Request paneli): GET metodu içeren metin kutusu.
- `Response Parser` (Request paneli): Saat Bazlı içeren metin kutusu.
- `Rota` (Request paneli): https://www.usom.gov.tr/url-list.txt içeren metin kutusu.
- `Headers` (Request paneli): Yok içeren metin kutusu.
- `Sayfalama` (Request paneli): Yok içeren metin kutusu.
- `Path` (JSON Path panelindeki Path alanı): JSON Path'ı girilen alan.
- `Response` (JSON Path Response panelindeki Response alanı): İstek gönderildikten sonra gelen yanıtın gösterileceği alan.
- `Henüz bir istek gönderilmedi.` (Yanıt): Bu alan, isteğin gönderildiği durumda kullanıcıya bilgi verir. Eğer istek gönderilirse ve yanıt alırsa, bu alanın içeriği değişecektir.

## Görünür kontroller

- `Yenile`: Refreshes the list.
- `Ekle`: Ekle
- `Göster/Gizle`: Shows or hides columns in the table.
- `Tamam`: Tamam
- `Filtrele`: Applies filters to the data sources list.
- `Filtreyi Temizle`: Clears all applied filters.
- `varsayılan`: Kullanılacak Arayüz seçeneği.
- `Çalışma Periyodu`: Saatlik, günlük gibi seçenekler arasından seçim yapmak için kullanılan düşme menü.
- `>`: URL ve Rota alanlarını arasında geçiş yapmak için kullanılan düğmeler.
