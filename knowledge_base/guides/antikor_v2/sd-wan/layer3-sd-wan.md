# Layer3 SD-WAN

## Kapsam

Layer 3 SD-WAN, OSI modelinin ağ katmanında (Layer 3) çalışan, IP tabanlı yönlendirme ile uzak ofisleri, veri merkezlerini veya bulut ortamlarını birbirine bağlayan yazılım tanımlı bir ağ teknolojisidir.

## Menü yolu

- `Sunucu Modu`

## Kullanım adımları

1. Yenile butonuna tıklayın.
2. Ekle butonuna tıklayın.
3. Durum seçeneğini aktif olarak işaretleyin veya bırakın.
4. Bağlantı adını girin veya seçin.
5. Açıklama alanına açıklamanızı girin veya seçin.
6. IPSec VPN profili seçeneğini belirleyin.
7. IPv4 veya IPv6 seçeneğini belirleyin.
8. IPSec kaynak IP adresini girin veya seçin.
9. Hedef port numarasını girin.
10. Durum: Aktif
11. Bağlantı Adı: [girilecek]
12. Açıklama: [girilecek]
13. Bağlantı Önceliği: [1-10 arasında bir sayı]
14. IPSec VPN Profili: [seçim]
15. IP Versiyon: IPv4
16. IPSec Kaynak IP Adresi: [seçim]
17. IPSec Hedef Dış IP Adresi: [girilecek]
18. Mobike: Aktif
19. Hedef Port: 4500

## Alanlar

- `Göster/Gizle` (Panel): Liste gösterimi seçimi (gizli veya gösteri)
- `Sayfa Başı Kayıt Sayısı` (Panel): Sayfada görüntülenecek kayıtların sayısı
- `Tamam` (Panel): Tüm kayıtları gösterme seçimi
- `Filtrele` (Panel): Filtreleme seçimi
- `Filtreyi Temizle` (Panel): Filtre temizleme seçimi
- `IP Versiyon` (IP Versiyon): IPv4 veya IPv6 seçeneğini belirler.
- `IPSec Kaynak IP Adresi` (IPSec Kaynak IP Adresi): IPSec kaynak IP adresini girin veya seçin.
- `IP Havuzu Başlangıç` (IP Havuzu Başlangıç): IP havuzunun başlangıcı için IPv4 adresi girin.
- `IP Havuzu Bitiş` (IP Havuzu Bitiş): IP havuzunun bitiş noktası için IPv4 adresi girin.
- `Hedef Port` (Hedef Port): Bağlantı için hedef port numarasını girin.
- `Bağlantı Adı` (Bağlantı Adı): Bağlantının adını girin.
- `Açıklama` (Açıklama): Bağlantı hakkında ek bilgileri girin.
- `Bağlantı Önceliği` (Bağlantı Önceliği): Bağlantının öncelik değerini girin (1-10).
- `IPSec VPN Profili` (IPSec VPN Profili): IPSec VPN profili seçin.
- `IPSec Hedef Dış IP Adresi` (IPSec Hedef Dış IP Adresi): IPSec hedef dış IP adresini girin.
- `Mobike` (Mobike): Mobike özelliğini etkinleştirin veya devre dışı bırakın.

## Görünür kontroller

- `Yenile`: Listeyi yeniden yükleme
- `Ekle`: Yeni kayıtlar ekleme
- `Durum`: Bağlantının durumu aktif olup olmadığını belirler.
- `Bağlantı Adı`: Bağlantının adını girin veya seçin.
- `Açıklama`: Bağlantının açıklamasını girin veya seçin.
- `IPSec VPN Profili`: IPSec VPN profili seçin.
- `İptal`: Ekrandaki işlemleri iptal etmek için kullanılır.
- `Kaydet`: Bağlantı bilgilerini kaydetmek için kullanılır.

## Uyarılar

- BGP Ayarları sayfasından ASN ve Paylaşılan Ağılar yapılandırmasını tamamlamayı unutmayın.
