# Proxy Ayarları

## Kapsam

Proxy internete erişim aşamasında kullanılan bir ara (vekil) sunucudur. Bir web sayfasına erişim sağlama aşamasında, isteğimiz vekil sunucuya gider, daha sonra vekil sunucu o WEB sayfasına bağlanarak içeriği alır, bize iletir. Proxy servis ayarlarından istemcilerin kullanacağı proxy IP adresi ve portu belirlenir. Ayrıca kullanıcıların Web Filtreleme Profilleri bölümünde tanımlanan politikalara dahil olup olmayacağı da seçilebilir.

## Kullanım adımları

1. Port Numarası alanına proxy sunucunun port numarasını girin.
2. Çıkış IP Adresi alanına proxy sunucunun çıkış IP adresini girin.
3. Web Filtreleme seçeneğini etkinleştirin veya devre dışı bırakın.
4. Proxy Otomatik Konfigürasyon (PAC) Hizmet IP Adresi alanına proxy sunucunun PAC hizmetinin IP adresini girin.

## Alanlar

- `Port Numarası` (Genel Ayarlar): Proxy sunucunun port numarasını girin.
- `Çıkış IP Adresi` (Genel Ayarlar): Proxy sunucunun çıkış IP adresini girin.
- `Web Filtreleme` (Genel Ayarlar): Web filtreleme politikalarını etkinleştirin veya devre dışı bırakın.
- `Proxy Otomatik Konfigürasyon (PAC) Hizmet IP Adresi` (Genel Ayarlar): Proxy sunucunun PAC hizmetinin IP adresini girin.

## Görünür kontroller

- `Yenile`: Ekranı yeniden yüklemek için kullanılır.
- `+ Ekle`: Yeni bir proxy ayarını eklemek için kullanılır.
