# DNS Ayarları

## Kapsam

DNS (Domain Name System), internet ortamında alan adından IP’ye, IP’den alan adına dönüşümlerini gerçekleştiren sistemdir. Sistemin kullandığı DNS bilgileridir. Yerel ağdaki bilgisayarlara da DHCP servisi üzerinden DNS bilgileri dağıtılır.

## Menü yolu

- `DNS Ayarları > DNS Ayarı - Yeni Kayıt`
- `Sistem Ayarlari > DNS Ayarları`

## Kullanım adımları

1. Ekle butonuna tıklayarak DNS adreslerini IPv4 veya IPv6 olarak ekleyin.
2. Adres Ailesi seçeneğini IPv4 olarak ayarlayın.
3. Dns Adresi alanında DNS adresini girin (örneğin, 8.8.8.8).
4. Açıklama alanına DNS kaydına ait açıklama metnini girin (örneğin, google).
5. Kaydet düğmesine tıklayarak DNS kaydı kaydedin.
6. DNS Adresi alanında '8.8.8.8' değerini girin.
7. Açıklama alanına 'google' yazın.
8. Kaydet düğmesine tıklayarak değişiklikleri kaydedin.

## Alanlar

- `Sıra` (DNS Ayarları paneli): DNS kaydının sırasını belirten alan.
- `Dns Adresi` (DNS Ayarları paneli): DNS sunucusunun IP adresini belirten alan.
- `Açıklama` (DNS Ayarları paneli): DNS kaydının açıklamasını belirten alan.
- `Adres Ailesi` (panel: Adres Ailesi): IPv4 veya IPv6 adres ailesini seçmek için kullanılan seçenekler.

## Görünür kontroller

- `Yenile`: Ekrandaki verileri yeniden yükleme.
- `Ekle`: DNS adreslerini eklemek için kullanılan düğme.
- `Göster/Gizle`: Veri göstergelerinin görünümünü ayarlamak için kullanılan menü.
- `Tamam`: Ekrandaki işlemleri onaylamak için kullanılan düğme.
- `Filtrele`: Verileri filtrelemek için kullanılan düğme.
- `Filtreyi Temizle`: Filtreleri temizlemek için kullanılan düğme.
- `İptal`: Eklenecek DNS kaydı iptal etmek için kullanılan düğme.
- `Kaydet`: Eklenecek DNS kaydı kaydetmek için kullanılan düğme.

## Uyarılar

- DNS kayıtları XLS, CSV veya PDF formatında indirilebilir durumdadır.
- Eğer IPv6 seçeneğini kullanmak isterseniz, DNS Adresi alanında IPv6 adresini girin.
- Model aynı etiketi control ve field olarak sınıflandırdı; yapılandırma değeri olan field korundu: Adres Ailesi
