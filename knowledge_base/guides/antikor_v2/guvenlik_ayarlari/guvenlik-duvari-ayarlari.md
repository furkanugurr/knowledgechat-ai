# Güvenlik Duvarı Ayarları

## Kapsam

Bu sayfa, trafik normalizasyon ayarlarını yapılandırmak için kullanılır.

## Kullanım adımları

1. Trafik Normalizasyonu seçeneğini etkinleştirmek için 'Açık' seçeneğine tıklayın.
2. Parçalanmış paketleri birleştirmek için 'Açık' seçeneğine tıklayın.
3. Rastgele IP identifikatörünü etkinleştirmek için 'Açık' seçeneğine tıklayın.
4. TCP normalizasyonunu etkinleştirmek için 'Açık' seçeneğine tıklayın.

## Görünür kontroller

- `Trafik Normalizasyonu`: Gelen ve giden paketlerin normalleştirilmesini etkinleştirir veya devre dışı bırakır.
- `Logla`: Normalleştirilmiş paket bilgilerinin loga kaydedilip kaydedilmeyeceği seçeneğidir.
- `Parçalanmış Paketleri Birleştir`: Parçalanmış paketlerin normalleştirilmesini etkinleştirir veya devre dışı bırakır.
- `Rastgele IP ID`: Rastgele IP identifikatörünün normalleştirilmesini etkinleştirir veya devre dışı bırakır.
- `TCP Normalizasyonu`: TCP paketlerinin normalleştirilmesini etkinleştirir veya devre dışı bırakır.
