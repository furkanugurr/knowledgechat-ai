# Network Arayüzleri Bant Genişiliği Monitörü

## Kapsam

Ağ arayüzlerine ait (DMZ, WAN, LAN ve VLAN) bant genişliği kullanımının anlık olarak izlenebildiği bu bölüm, her arayüz üzerinden geçen veri trafiğini gerçek zamanlı olarak görüntülemeye olanak tanır. Bu sayede yöneticiler, ağ trafiğini analiz edebilir, olası yoğunlukları tespit edebilir ve kaynak kullanımını daha verimli bir şekilde yönetebilir.

## Alanlar

- `Arayüz Adı` (Network Arayüzleri Bant Genişiliği Monitörü paneli): Ağ arayüzü adını gösterir. Örneğin: ix0-LAN1, ix1-WAN1, etc.
- `İndirme Bantgenişliği` (Network Arayüzleri Bant Genişiliği Monitörü paneli): Ağ arayüzü üzerinden indirilen veri miktarını gösterir. Örneğin: 0 bit, 1 Mbit, 264 Kbit.
- `Yükleme Bantgenişliği` (Network Arayüzleri Bant Genişiliği Monitörü paneli): Ağ arayüzü üzerinden yüklenecek veri miktarını gösterir. Örneğin: 0 bit, 1 Mbit, 202 Kbit.
- `İndirme Paket Oranı` (Network Arayüzleri Bant Genişiliği Monitörü paneli): Ağ arayüzü üzerinden indirilen paketlerin sayısı. Örneğin: 0, 68544, 13149.
- `Yükleme Paket Oranı` (Network Arayüzleri Bant Genişiliği Monitörü paneli): Ağ arayüzü üzerinden yüklenecek paketlerin sayısı. Örneğin: 0, 68544, 17157.
