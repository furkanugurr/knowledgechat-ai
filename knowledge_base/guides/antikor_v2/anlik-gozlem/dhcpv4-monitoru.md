# DHCPv4 Monitörü

## Kapsam

DHCPv4 servisi tarafından dağıtılan IPv4 adresleri alan cihazların bilgilerini anlık olarak gösteren bölümdür.

## Menü yolu

- `DHCPv4 Monitörü`

## Kullanım adımları

1. IP Adresi
2. MAC Adresi
3. Network
4. Kayıtlı Bilgisayar Adı
5. Görünen Bilgisayar Adı
6. Kira Başlangıcı
7. Kira Bitişi

## Alanlar

- `IP Adresi` (192.168.100.10): DHCP tarafından dağıtılan IP adresini gösterir.
- `MAC Adresi` (:ba:75:1c:): DHCP tarafından dağıtılan MAC adresini gösterir.
- `Network` (LAN1 - LAN1): DHCP ağ bilgisini gösterir.
- `Kayıtlı Bilgisayar Adı` (sunucu): DHCP tarafından kaydedilen bilgisayar adını gösterir.
- `Görünen Bilgisayar Adı` (desktop-5qq7l): DHCP tarafından görünen bilgisayar adını gösterir.
- `Kira Başlangıcı` (2020-04-03 06:21:23): DHCP tarafından verilen kira başlangıç tarihini ve saatini gösterir.
- `Kira Bitişi` (2020-04-18 12:22:46): DHCP tarafından verilen kira bitiş tarihini ve saatini gösterir.
- `Toplam Kira Süresi` (1 day): DHCP tarafından verilen kira süresini gösterir.

## Görünür kontroller

- `XLS`: Excel dosyası oluştur.
- `CSV`: CSV dosyası oluştur.
- `PDF`: PDF dosyası oluştur.
- `Listeleme Türü`: Listeleme türünü seçmek için kullanılan menü.
- `Göster/Gizle`: Gösterilen bilgileri gizlemek veya göstermek için kullanılan düğme.
- `Filtrele`: Filtreleme menüsü açmak için kullanılan düğme.
- `Sil`: Kayıtları silmek için kullanılan düğme.

## Uyarılar

- Filtreleme menüsüne tıklayarak IPv4 adresi, MAC adresi, network bazlı vb. gibi filtreleme yapılabilmektedir.
- Model aynı etiketi control ve field olarak sınıflandırdı; yapılandırma değeri olan field korundu: Görünen Bilgisayar Adı, IP Adresi, Kayıtlı Bilgisayar Adı, Kira Başlangıcı, Kira Bitişi, MAC Adresi, Network
