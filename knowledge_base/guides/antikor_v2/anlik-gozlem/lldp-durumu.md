# LLDP Durumu

## Kapsam

LLDP Ayarları yapıldıktan sonra LLDP durumu bu menüden gözlemlenebilmektedir.

## Kullanım adımları

1. LLDP ayarları yapıldıktan sonra LLDP durumu bu menüden gözlemlenebilmektedir.

## Alanlar

- `Interface` (LLDP neighbors): LLDP üzerinden alınan interface bilgisi
- `ChassisID` (Chassis): Cihazın chassis ID'si
- `SysName` (Chassis): Sistem adı
- `SysDescr` (Chassis): Sistem açıklaması
- `MgmtIP` (Chassis): Kurumsal IP adresi
- `Capability` (Chassis): Yetenekler
- `PortID` (Port): Port ID'si
- `PortDescr` (Port): Port açıklaması
- `TTL` (Port): Time to Live
