# DHCPv6 Monitörü

## Kapsam

DHCPv6 tarafından IPv6 adresleri alan cihazların bilgilerini anlık olarak gösteren bölümdür.

## Menü yolu

- `DHCPv6 Monitörü`

## Kullanım adımları

1. Listeleme Türü: Anlık Gösterim
2. Filtreleme: IPv6 adresi, MAC adresi, network bazlı vb. gibi filtreleme yapılabilmektedir.
3. IPv4 adresini girin veya boş bırakın.
4. MAC adresini girin veya boş bırakın.
5. Network seçeneğini kullanarak filtre ayarlayın.
6. Kayıtlı Bilgisayar Adını girin veya boş bırakın.
7. Görünen Bilgisayar Adını girin veya boş bırakın.
8. Kira Başlangıcı tarihini belirleyin veya boş bırakın.
9. Kira Bitişi tarihini belirleyin veya boş bırakın.
10. Filtrele butonuna basarak filtre uygulayın.

## Alanlar

- `IP Adresi` (IP Adresi sütunu başlığı): IP adresi bilgisi girilir veya seçilir
- `MAC Adresi` (MAC Adresi sütunu başlığı): MAC adresi bilgisi girilir veya seçilir
- `Network` (Network sütunu başlığı): Network bilgisi girilir veya seçilir
- `Kayıtlı Bilgisayar Adı` (Kayıtlı Bilgisayar Adı sütunu başlığı): Kayıtlı bilgisayar adı bilgisi girilir veya seçilir
- `Görünen Bilgisayar Adı` (Görünen Bilgisayar Adı sütunu başlığı): Görünen bilgisayar adı bilgisi girilir veya seçilir
- `Kira Başlangıcı` (Kira Başlangıcı sütunu başlığı): Kira başlangıç tarihi bilgisi girilir veya seçilir
- `Kira Bitişi` (Kira Bitişi sütunu başlığı): Kira bitiş tarihi bilgisi girilir veya seçilir
- `Toplam Kira Süresi` (Toplam Kira Süresi sütunu başlığı): Toplam kira süresi bilgisi girilir veya seçilir
- `IPv4` (IPv4): IP Adresi
- `MAC` (MAC): MAC Adresi
- `Filtre Yok` (Network): Network
- `Tarihinde` (Kira Başlangıcı): Kira Başlangıcı

## Görünür kontroller

- `Listeleme Türü`: Anlık Gösterim
- `Yenile`: Listeyi yeniden yükle
- `Filtrele`: Filtre uygula
