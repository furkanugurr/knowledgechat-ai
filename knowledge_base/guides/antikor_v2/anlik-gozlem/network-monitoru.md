# Network Monitörü

## Kapsam

Anlık tüm trafiğin tcpdump parametreleri, belli filtreler ve/veya Layer bazında görüntülenebildiği sayfadır.

## Kullanım adımları

1. Başlat
2. Ternizle

## Alanlar

- `Çıktı` (Parametreler): Ekran veya PCAP olarak kaydetme.
- `DNS Çözümlemesi` (Parametreler): Yapın veya Yapmasın.
- `Ethernet Başlıkları` (Parametreler): Göster veya Gösterme.
- `Detaylı Sonuç` (Parametreler): Göster veya Gösterme.
- `Hazır Filtreler` (Parametreler): Filtre seçenekleri.
- `Protokol` (Parametreler): Protokol seçimi.
- `Ek Parametreler` (Parametreler): Ek parametreler.
- `Snapshot Boyutu` (Parametreler): Snapshot boyutu.

## Görünür kontroller

- `Başlat`: Network Monitörü başlatma.
- `Ternizle`: Network Monitörü temizleme.

## Uyarılar

- En üsteki kayıt en son gelen çıktıdır. Çıktı geçmişinde maksimum 25 satır gösterilir.
