# Port Yönlendirme

## Kapsam

Port yönlendirme, yerel ağdaki sunuculara internet üzerinden bağlantı kurulabilmesi için ihtiyaç duyulan portları ilgili IP adresine yönlendirme işlemidir. TCP, UDP vb. protokolleri desteklemektedir. Yönlendirmesi yapılan özel IP adresi ve portları internet üzerinden girildiği zaman, sistem (Antikor) IP adresine karşılık gelen yerel ağdaki IP adresine ve porta yönlendirme gerçekleştirmektedir.

## Menü yolu

- `Port Yönlendirme`

## Kullanım adımları

1. Aktif durumda olan port yönlendirmesini görüntüleyin.

## Alanlar

- `Göster/Gizle` (Port Yönlendirme paneli): Liste gösterimini gizle veya göster seçeneği.
- `Sayfa Başı Kayıt Sayısı` (Port Yönlendirme paneli): Sayfada görüntülenecek kayıtların sayısı.

## Görünür kontroller

- `Yenile`: Listeyi yeniden yükle.
- `+ Ekle`: Yeni bir port yönlendirme kaydı ekleyin.
