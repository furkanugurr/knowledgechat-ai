# NAT Havuzları

## Kapsam

NAT IP adresleri bu menüde tanımlanmaktadır.

## Menü yolu

- `NAT Havuzları`

## Kullanım adımları

1. Adı: LAN2 Çıkış
2. IP Adresleri: 192.168.168.68
3. Açıklama: LAN2 Çıkış
4. İşlemler: Düzenle, Sil
5. Adı alanına NAT havuzu adını girin.
6. IP Adresleri seçeneğini kullanarak IP adreslerini belirleyin.
7. Açıklama alanına NAT havuzu hakkında açıklama yapın.
8. Kaydet butonuna tıklayarak kaydınızı kaydedin.

## Alanlar

- `Adı` (Adı): NAT havuzu adı
- `IP Adresleri` (IP Adresleri): NAT havuzunun IP adresi
- `Açıklama` (Açıklama): NAT havuzu açıklaması

## Görünür kontroller

- `Yenile`: Menüyü yeniden yükler
- `Ekle`: Yeni NAT havuzu ekleme
- `İptal`: Kaydını iptal etmek için.
- `Kaydet`: Girilen bilgileri kaydetmek için.

## Uyarılar

- Adı ve IP Adresleri alanları boş bırakılamaz.
