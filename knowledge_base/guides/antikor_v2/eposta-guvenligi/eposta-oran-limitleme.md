# E-Posta Oran Limitleme

## Kapsam

Bu menü, e-posta için belirli kriterlerde limitleme imkanı sağlamakta olup aynı zamanda IP Beyaz Listesi ve Kullanıcı Beyaz Listesi içermektedir.

## Menü yolu

- `E-Posta Güvenliği > E-posta Oran Limitleme`
- `E-Posta Oran Limitleme > IP Beyaz Listesi - Yeni Kayıt`
- `IP Beyaz Listesi > Yeni Kayıt`
- `E-Posta Oran Limitleme`

## Kullanım adımları

1. Durum seçeneğini belirleyin.
2. IP / Network Adresi alanına IP adresini girin.
3. Açıklama alanına gerekli bilgileri girin.
4. Kaydet butonuna tıklayın.
5. Aktif durumu seçin veya bırakın.
6. IP adresi, IP aralığı, ağ tanımları veya GEO IP adreslerini eklemek için uygun butonu tıklayın.

## Alanlar

- `Durum` (Limit Kriteri panelinde): Limit Kriteri için durumu gösteren checkbox
- `Burst` (Limit Kriteri panelinde): Limit Kriteri için burst değerini gösteren field
- `Limit Oranı` (Limit Kriteri panelinde): Limit Kriteri için limit oranını gösteren dropdown
- `IP / Network Adresi` (IP Beyaz Listesi - Yeni Kayıt panelinde): Yeni IP Beyaz Listesi kaydı oluşturmak için IP adresini girin.
- `Açıklama` (IP Beyaz Listesi - Yeni Kayıt panelinde): Kayıtlı IP adresine ait açıklama yapın.
- `IP / Network Adresi` (Durum panelinde): Beyaz listeye eklenen IP adresi veya ağ aralığı bilgisi girilir.
- `Kullanıcı` (Durum altındaki alan): E-posta adresi veya kullanıcı adı girilmesi gereken alan
- `Açıklama` (Durum altındaki alan): Kullanıcı hakkında ek bilgi girişi yapmak için kullanılan alan

## Görünür kontroller

- `Yenile`: IP Beyaz Listesi için yeniden yükleme
- `Ekle`: IP Beyaz Listesi için ekleme
- `Durum`: Kayıtlı IP adresinin aktif olup olmadığını belirler.
- `İptal`: Ekrandaki işlemleri iptal etmek için kullanılır.
- `Kaydet`: Kayıtları kaydetmek için kullanılır.
- `Durum`: Kayıtın aktif olup olmadığını belirler.
- `IP Adresi`: Belli bir IP adresini beyaz listeye eklemek için kullanılır.
- `IP Aralığı`: Belli bir IP aralığını beyaz listeye eklemek için kullanılır.
- `Ağ Tanımları`: Ağ tanımlarını beyaz listeye eklemek için kullanılır.
- `Geo IP`: GEO IP adreslerini beyaz listeye eklemek için kullanılır.
- `Durum`: Kullanıcıların durumu aktif olarak işaretlenmesi
- `İptal`: E-posta oran limitleme işleminden çıkartmak için kullanılan düğme
- `Kaydet`: Kullanıcıları beyaz listeye kaydetmek için kullanılan düğme
