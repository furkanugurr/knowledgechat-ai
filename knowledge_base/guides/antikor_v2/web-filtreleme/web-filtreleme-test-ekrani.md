# Web Filtreleme Test Ekranı

## Kapsam

Web Filtreleme Profilleri bölümünde tanımlanmış (engelli/izinli) olan siteler için test işleminin yapıldığı bölümdür.

## Kullanım adımları

1. Adres Ailesi seçeneğini IPv4 olarak belirleyin veya IPv6 seçeneğini kullanarak IP adresinin ailesini belirleyin.
2. Kaynak IP Adresi alanına gerekli IP adresini girin.
3. Test Edilecek URL alanına test etmek istediğiniz URL'yi girin.
4. URL Test Et düğmesine tıklayarak test işlemini başlatın.

## Alanlar

- `Kaynak IP Adresi` (IPv4): Test edilecek URL'ye ait kaynak IP adresini girmek için kullanılan alan. IPv4 ve IPv6 seçenekleri mevcuttur.
- `Test Edilecek URL` (http://): Test edilecek URL'yi girerek test etmek için kullanılan alan.

## Görünür kontroller

- `Adres Ailesi`: IP adresinin ailesini seçmek için kullanılan seçenekler. IPv4 ve IPv6 seçenekleri mevcuttur.
- `URL Test Et`: Test edilecek URL'yi test etmek için kullanılan düğme.

## Uyarılar

- URL Test Et düğmesi ile test işlemi başlatılabilir.
