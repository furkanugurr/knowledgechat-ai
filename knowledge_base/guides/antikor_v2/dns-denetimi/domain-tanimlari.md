# Domain Tanımları

## Kapsam

Bu bölüm, DNS tanımlarının bulunduğu yerdir. Belli etki alanlarının her zaman belli Alan Adı Sunucularından (Domain Name Server) sorgulanması isteniyorsa bu alana tanımlanması gerekmektedir.

## Menü yolu

- `Domain Tanımları`

## Uyarılar

- Active Directory kullanılan ağlarda Antikor’un DNS Filtreleme özelliği kullanılabilmesi için, Active Directory etki alanını ve dns sunucusu bu ekrandan tanımlanmalıdır.
