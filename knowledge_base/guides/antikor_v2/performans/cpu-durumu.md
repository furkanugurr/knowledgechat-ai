# CPU Durumu

## Kapsam

CPU modeli ve çekirdek sayısını gösterdiği gibi, seçilen tarih aralığındaki CPU kullanım istatistiklerini de sunmaktadır. Bu sayfa, belirli bir zaman diliminde CPU’nun kullanım verilerini ayrıntılı bir şekilde sağlar.

## Kullanım adımları

1. Başlangıç ve Bitiş tarihlerini girmek için 'Göster' butonuna tıklayın.

## Alanlar

- `Başlangıç Tarihi` (CPU Durumu paneli): CPU kullanım verilerinin başlangıcı tarihini belirtir.
- `Bitiş Tarihi` (CPU Durumu paneli): CPU kullanım verilerinin bitiş tarihini belirtir.

## Görünür kontroller

- `Göster`: CPU kullanım verilerini gösterir.
